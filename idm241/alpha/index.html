Alpha - Information

Trigger

  • Mouse hover overs the image
  • The hover will be active or inactive over the image

Rules

  • Gold background 50% transparency moves from bottom to top
  • Text and incon inward from the bottom
  • Gold background, there's text and a button
  • On hover, the cursor changes from default to pointer

Feedback

  • On hover, the cursor changes from default to pointer
  • Gold background, there's text and a button
  • User may feel the hover's decelerates ease-in
  • Touching the image will cause it to automatically zoom in
  • Offing the hover effect will have the reverse effect

Loops and Modes
