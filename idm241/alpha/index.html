Microinteractions - IDM241

Alpha - Information

Trigger

  • Hover Will activate when the mouse touchs the image
  • The hover will be active or inactive over the image

Rules

  • Saturation color will be on hover
  • Text and incon inward from the bottom
  • On click icon will direct you to a page
  • Cursor changes to pointer on touch the icon

Feedback

  • The user will notice a pointer instead of a cursor
  • User can observe text or an icon move from the bottom to the middle
  • User may feel the hover's decelerates ease-in
  • Toucuhing the image will cause it to automatically zoom in
  • Offing the hover effect will have the reverse effect

Loops and Modes

  • None
