Microinteractions - IMD241

Alpha - Information

Triggers

  • Touch

  • Hover over image (On/Off Hover)

Rules

  • Apply saturation color on hover

  • Text and icon inward from bottom

  • Click icon direct you to a page

  • Cursor changes to pointer on icon

Feedback

  • Text/Icon slide towards center from bottom

  • User see the image zoom in

  • User can feel the decelerates ease in hover

  • Off Hover reverse the effect

  • Cursor changes to pointer on icon

Loops and Modes

  • None
